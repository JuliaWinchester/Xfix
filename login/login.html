AnatDB is an example site demonstrating Xfix, an open source web app for anatomical education. This app lists anatomical specimens (e.g., bones or realistic plastic models) sorted by chapter or topic. Users can view multiple images of each specimen. Images can be labelled to show specimen structures or regions. For each specimen, it is also possible to see all other specimens with similar structures. The app administrator has the ability to create, edit, and delete chapters, specimens, image views, and labels.
